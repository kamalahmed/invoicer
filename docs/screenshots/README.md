# Screenshots

The main `README.md` references four images from this folder:

| Filename | What to capture |
|---|---|
| `dashboard.png` | The Dashboard view with a few saved invoices and clients. |
| `editor.png` | The Editor view with the sample invoice on the right. |
| `templates.png` | The Template & branding section open, showing the picker. |
| `mobile.png` | The app at a phone width (e.g. 375 × 812), Editor tab selected. |

Capture from the [live demo](https://invoicer-one-zeta.vercel.app/) and drop the
files in this folder. Recommended dimensions: 2× the displayed size for
crisp rendering on Retina displays (e.g. 2400 × 1500 for the dashboard).

Format `.png` with light compression. Keep each image under ~500 KB.
